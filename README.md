# Portfolio Website

Modern portfolio website for an Electronics, Communication & Information Engineering student at IOE Pulchowk Campus. Features a responsive design showcasing projects in AI/ML, Robotics, and IoT.

## Key Technical Features

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4 with custom animations
- **UI Components**: Radix UI + shadcn/ui component library
- **Deployment**: Static export enabled (Cloudflare Pages compatible)
- **Analytics**: Vercel Analytics integration
- **Content**: Projects showcase (Self-Balancing Robot, ParkSense IoT, Setu AI Legal Assistant)
